# 电子板块价值评分公式 (Electronics Sector Value Score Formula)

## 概述

本公式是A股电子板块（申万一级：电子，东方财富板块代码BK1201）量化价值评分体系。
评分范围0-100分，分数越高表示该股票越具有投资价值。

**核心设计理念：**
1. ROE(12%)为主锚指标 — 高质量电子企业维持15%+ ROE
2. 毛利率(7%)为行业特色指标 — Fabless芯片设计50%+毛利率 vs 封测/组装15-25%，差异化显著
3. 成长性权重(25%)为全系列最高 — 科技股增长是核心估值驱动力
4. 估值权重(18%)偏低 — 半导体周期性导致PE波动大，降低PE权重避免失真
5. 18个非零权重指标参与回测，100%权重覆盖
6. 评分规则全套应用：增长率±200%封顶、ROE/ROIC≤60%封顶、PE<0排除、覆盖率惩罚

**行业特点：**
- 503只股票，A股第二大行业（仅次于医药生物506只）
- 113只亏损股（22.5%），远高于食品饮料（13%）和家电（11%）
- 半导体周期性强，PE波动剧烈（从10x到500x+）
- R&D支出是核心竞争力但API缺乏研发数据（已记录为数据缺口）
- 57只股票利润增速被±200%封顶（11.3%，全系列最高）

## 第三部分：公式

```
Value_Score = base_score * coverage_penalty
base_score = Sum(weight_i * score_i) / Sum(weight_i_participated)
score_i = industry_rank_percentile (0-100)
  越低越好: score = 100 - percentile
  越高越好: score = percentile
  percentile = (rank - 1) / (N_valid - 1) * 100

Scoring rules:
  ROE cap: min(ROE, 60%)
  ROIC cap: min(ROIC, 60%)
  Growth cap: clip(growth, -200%, +200%)
  PE: loss-making (PE<0) excluded from PE ranking
  Coverage penalty: if indicators < 10, score *= (count / 18)
  Missing: excluded from that indicator, weight redistributed
```
