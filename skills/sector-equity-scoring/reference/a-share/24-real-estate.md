# 房地产板块价值评分公式 (Real Estate Sector Value Score Formula)

## 概述

本公式是A股房地产板块（申万一级：房地产，东方财富板块代码BK0451）量化价值评分体系。
评分范围0-100分，分数越高表示该股票越具有投资价值。

**核心设计理念：**
1. ROE(12%)为主锚指标 — 房地产行业ROE极度分化（南京高科12.7% vs 京投发展-376.7%），高ROE是存活者标志
2. 股息率权重(9%)为安全边际锚 — 存活优质房企分红稳定（中国国贸5.00%、津滨发展4.48%、南京高科3.49%），类似煤炭板块的价值锚定逻辑
3. 成长性权重仅12% — 行业整体萎缩，18只利润增速被±200%封顶（24%），2只营收增速被封顶，降权避免过度惩罚衰退
4. 现金流权重(15%)高配 — 房企流动性危机频发（恒大/碧桂园），现金流质量是生死分界线
5. PB(6%)对房企重要 — 37/75只PB<1x（49.3%为破净股），资产折价是价值信号
6. 风险类(13%)适度偏高 — D/A中位数67.4%为行业常态，但极端杠杆（>90%）暗示违约风险
7. 18个非零权重指标参与回测，100%权重覆盖
8. V2规则全套应用：增长±200%封顶、ROE/ROIC≤60%封顶、PE<0排除、覆盖率惩罚

**行业特点：**
- 75只股票（排除8只ST/*ST + 9只B股），涵盖住宅开发/商业地产/产业园区/城投等子行业
- 37只亏损股（49.3%），为全系列最高亏损率之一，反映行业深度出清
- 46只无股息（61.3%），大量房企停止分红
- PE中位数43.7（仅正PE股票），PB中位数1.02
- 37只破净股（PB<1x，占49.3%），行业资产大幅折价
- D/A中位数67.4%（范围6.4%-102.6%），高杠杆为行业常态
- 18只利润增速被±200%封顶（24%），2只营收增速被封顶
- 0只ROE/ROIC被60%封顶（最高ROE仅12.7%，行业盈利能力极弱）
- 万科(#63)、金地(#53)、绿地(#68)排名靠后反映其深度困境状态
- 中国国贸(#1)、南京高科(#2)等非住宅开发类房企凭借盈利/分红优势领先

## 第三部分：公式

```
Value_Score = base_score * coverage_penalty
base_score = Sum(weight_i * score_i) / Sum(weight_i_participated)
score_i = industry_rank_percentile (0-100)
  越低越好: score = (1 - (rank - 1) / (N_valid - 1)) * 100  (rank ascending)
  越高越好: score = (1 - (rank - 1) / (N_valid - 1)) * 100  (rank descending)

V2 rules (applied BEFORE rank_percentile):
  ROE cap: min(ROE, 60%)
  ROIC cap: min(ROIC, 60%)
  Growth cap: clip(growth, -200%, +200%)
  PE: loss-making (PE<0) excluded from PE ranking
  Coverage penalty: if indicators < 10, score *= (count / 18)
  Missing: excluded from that indicator, weight redistributed

Weight table:
  PE:         5% (low好)    PB:        6% (low好)    股息率:     9% (high好)
  ROE:       12% (high好)   ROA:       3% (high好)   ROIC:       5% (high好)
  毛利率:     5% (high好)   净利率:     5% (high好)
  营收增长:    5% (high好)   利润增长:   7% (high好)
  CF/净利润:   5% (high好)   FCF收益率:  5% (high好)   CF/营收:    5% (high好)
  资产负债率:  5% (low好)    流动比率:   4% (high好)   带息负债率:  4% (low好)
  毛利率变化:  5% (high好)   速动比率:   5% (high好)
```
