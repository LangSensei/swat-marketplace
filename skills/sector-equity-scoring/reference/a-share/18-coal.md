# 煤炭板块价值评分公式 (Coal Sector Value Score Formula)

## 概述

本公式是A股煤炭板块（申万一级：煤炭，东方财富板块代码BK0437）量化价值评分体系。
评分范围0-100分，分数越高表示该股票越具有投资价值。

**核心设计理念：**
1. ROE(12%)为主锚指标 — 煤炭行业ROE分化显著（陕煤13.7%、神华12.8% vs 亏损股-65.9%），高ROE代表持续盈利能力
2. 股息率权重(9%)为行业最高 — 煤炭是A股高分红板块代表（冀中能源10.87%、平煤7.30%、恒源煤电6.23%），高股息率是价值投资核心安全边际
3. 成长性权重(15%)适度降低 — 国企主导+周期下行期，多数公司营收/利润同比负增长，降低权重避免过度惩罚周期底部
4. 现金流权重(12%)标准偏高 — 资本密集型采矿业，现金流质量区分真盈利与纸面利润
5. 风险权重(12%)标准偏高 — 煤炭企业杠杆分化大（神华23.3% vs 郑州煤电81.9%），D/A+带息负债率有显著区分度
6. 净利率(5%)权重偏高 — 煤炭行业净利率分化极大（神华21.3% vs 亏损股-50.9%），比毛利率更能反映综合盈利
7. 18个非零权重指标参与回测，100%权重覆盖
8. 评分规则全套应用：增长率±200%封顶、ROE/ROIC≤60%封顶、PE<0排除、覆盖率惩罚

**行业特点：**
- 32只股票（排除1只ST后），涵盖动力煤/焦煤/煤化工等子行业
- 13只亏损股（40.6%），反映煤炭行业周期底部压力显著
- 国有企业主导，ROE中位数普遍较低（盈利股中位数约4-6%）
- PE中位数18.9（仅正PE股票），PB中位数1.14
- 2只利润增速被±200%封顶（郑州煤电、辽宁能源）
- 0只ROE/ROIC被60%封顶
- 高分红特征突出：冀中10.87%、平煤7.30%、恒源6.23%、山煤5.91%、昊华5.60%

## 第三部分：公式

```
Value_Score = base_score * coverage_penalty
base_score = Sum(weight_i * score_i) / Sum(weight_i_participated)
score_i = industry_rank_percentile (0-100)
  越低越好: score = (1 - (rank - 1) / (N_valid - 1)) * 100  (rank ascending)
  越高越好: score = (1 - (rank - 1) / (N_valid - 1)) * 100  (rank descending)

Scoring rules:
  ROE cap: min(ROE, 60%)
  ROIC cap: min(ROIC, 60%)
  Growth cap: clip(growth, -200%, +200%)
  PE: loss-making (PE<0) excluded from PE ranking
  Coverage penalty: if indicators < 10, score *= (count / 18)
  Missing: excluded from that indicator, weight redistributed

Weight table:
  PE:         7% (low好)    PB:        5% (low好)    股息率:     9% (high好)
  ROE:       12% (high好)   ROA:       3% (high好)   ROIC:       5% (high好)
  毛利率:     5% (high好)   净利率:     5% (high好)
  营收增长:    7% (high好)   利润增长:   8% (high好)
  CF/净利润:   5% (high好)   FCF收益率:  4% (high好)   CF/营收:    3% (high好)
  资产负债率:  4% (low好)    流动比率:   4% (high好)   带息负债率:  4% (low好)
  毛利率变化:  5% (high好)   速动比率:   5% (high好)
```
