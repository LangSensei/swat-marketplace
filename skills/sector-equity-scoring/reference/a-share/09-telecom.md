# 通信板块价值评分公式

## 概述

本公式涵盖130只通信板块股票，采用18指标百分位排名方法论。

**关键评分规则：**
1. **增长率封顶**：±200%增长率cap应用于评分计算，回测表格显示截断后值
2. **ROE封顶**：宁通信B ROE=105.3%（近零净资产极端值），封顶至60%
3. **覆盖率惩罚**：使用`cnt/18`公式（与家电/医药/食品饮料一致）。本数据集最少16指标，实际无影响
4. **显示一致性**：回测表格所有数值均显示应用评分规则后的值

**核心设计理念：**
- ROE(12%)为主锚指标
- 估值(20%)、盈利能力(30%)、成长(20%)、现金流(12%)、风险(10%)、行业质量(8%)
- 18个指标参与回测，100%权重覆盖
- 覆盖率惩罚：指标数<10的股票得分乘以(指标数/18)

## 公式

```
Value_Score = base_score * coverage_penalty
base_score = Sum(weight_i * score_i) / Sum(weight_i_participated)
score_i = industry_rank_percentile (0-100)
  lower_better: score = 100 - percentile
  higher_better: score = percentile
  percentile = (rank - 1) / (N_valid - 1) * 100

Scoring rules:
  ROE cap: min(ROE, 60%)
  ROIC cap: min(ROIC, 60%)
  Growth cap: clip(growth, -200%, +200%)
  PE: loss-making (PE<0) excluded from PE ranking
  Coverage penalty: if indicators < 10, score *= (count / 18)
  Missing: excluded from that indicator, weight redistributed
```

### 关键发现

1. **封顶规则影响极小**: 21只股票被截断，但最大得分变化仅±0.4分，Top 10完全不变
2. **光模块板块爆发**: AI算力需求驱动，中际旭创/天孚通信/新易盛稳居Top 10
3. **运营商三巨头稳健但平庸**: 低PE+高股息但低增长+低ROE，集中在#15–27区间
4. **高亏损率**: 37只(28%)亏损，反映设备商周期性
5. **ROE两极分化**: 亿联网络30.7% vs 运营商5–10%，ROE是最有效的区分指标

### 局限性

1. 使用YoY增长率而非3年CAGR—单年波动影响较大
2. 缺少研发投入强度指标—通信行业研发是核心竞争力
3. 缺少订单/合同负债数据—5G/AI订单能见度是重要前瞻指标
4. 纯横截面排名，无历史纵向对比
