# 机械设备板块价值评分公式 (Machinery Equipment Sector Value Score Formula)

## 概述

本公式是A股机械设备板块（申万一级：机械设备，东方财富板块代码BK1205）量化价值评分体系。
评分范围0-100分，分数越高表示该股票越具有投资价值。

**核心设计理念：**
1. ROE(12%)为主锚指标 — 通用设备/专用设备ROE中位数~8-10%，龙头可达20%+
2. 估值权重(18%)标准配置 — 机械设备PE波动大（通用设备vs自动化设备差异显著）
3. 成长性权重(22%)适中 — 自动化/机器人高增长，工程机械周期性，通用设备稳增长
4. 现金流权重(12%)偏高 — 资本密集型行业，现金流质量关键
5. 毛利率(7%)为行业特色指标 — 区分专用设备(高毛利40%+)和通用设备(低毛利20%)
6. 质量类(8%)中毛利率变化(4%)+速动比率(4%) — 关注盈利质量和偿债能力
7. 18个非零权重指标参与回测，100%权重覆盖
8. V2规则全套应用：增长率±200%封顶、ROE/ROIC≤60%封顶、PE<0排除、覆盖率惩罚

**行业特点：**
- 527只股票（排除74只B股+14只ST/*ST），涵盖五大子行业
- 92只亏损股（17.5%），分布于各子行业
- PE中位数61.3（需/100转换），PB中位数3.59
- 53只股票利润增速被±200%封顶（10.1%），0只营收增速被封顶
- 子行业分化显著：通用设备(41.2%最大)、专用设备(32.8%)、自动化设备(15.0%)

## 第三部分：公式

```
Value_Score = base_score * coverage_penalty
base_score = Sum(weight_i * score_i) / Sum(weight_i_participated)
score_i = industry_rank_percentile (0-100)
  越低越好: score = 100 - percentile
  越高越好: score = percentile
  percentile = (rank - 1) / (N_valid - 1) * 100

V2 rules:
  ROE cap: min(ROE, 60%)
  ROIC cap: min(ROIC, 60%)
  Growth cap: clip(growth, -200%, +200%)
  PE: loss-making (PE<0) excluded from PE ranking
  Coverage penalty: if indicators < 10, score *= (count / 18)
  Missing: excluded from that indicator, weight redistributed

Weight table:
  PE:         6% (low好)    PB:        5% (low好)    股息率:     7% (high好)
  ROE:       12% (high好)   ROA:       3% (high好)   ROIC:       5% (high好)
  毛利率:     7% (high好)   净利率:     3% (high好)
  营收增长:   10% (high好)   利润增长:  12% (high好)
  CF/净利润:   5% (high好)   FCF收益率:  4% (high好)   CF/营收:    3% (high好)
  资产负债率:  4% (low好)    流动比率:   3% (high好)   带息负债率:  3% (low好)
  毛利率变化:  4% (high好)   速动比率:   4% (high好)
```
