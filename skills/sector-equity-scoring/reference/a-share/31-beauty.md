# 美容护理板块价值评分公式 (Beauty & Personal Care Sector Value Score Formula)

## 概述

本公式是A股美容护理板块（申万一级：美容护理，东方财富板块代码BK1035）量化价值评分体系。
评分范围0-100分，分数越高表示该股票越具有投资价值。

**核心设计理念：**
1. ROE(15%)为主锚指标 — 美容护理龙头企业持续维持15-30%+ ROE，品牌护城河的直接体现
2. 毛利率(8%)为行业特色指标 — 化妆品/医美企业70-90%+毛利率是品牌壁垒核心
3. 成长性权重(20%)与食品饮料持平 — 美容护理行业仍处扩张期，增长驱动价值
4. 估值权重(20%)适中 — 高成长消费品允许合理溢价
5. 18个非零权重指标参与回测，100%权重覆盖

**V2规则：**
- 排除ST/*ST和B股
- 显示封顶后数值（ROE/ROIC ≤60%，增长率 ±200%）
- FCF收益率 = FCFF_BACK / f20（总市值），非f116
- 排名表包含"指标"列（X/18数据覆盖）
- 覆盖率惩罚：指标数<10的股票得分乘以(指标数/18)

## 第三部分：公式

```
Value_Score = base_score * coverage_penalty
base_score = Sum(weight_i * score_i) / Sum(weight_i_participated)

score_i = industry_rank_percentile (0-100)
  越低越好: score = 100 - percentile
  越高越好: score = percentile
  percentile = (rank - 1) / (N_valid - 1) * 100

V2 rules:
  ROE cap: min(ROE, 60%)
  ROIC cap: min(ROIC, 60%)
  Growth cap: clip(growth, -200%, +200%)
  PE: loss-making (PE<0) excluded from PE ranking
  Coverage penalty: if indicators < 10, score *= (count / 18)
  Missing: excluded from that indicator, weight redistributed
```

### 局限性

1. 使用YoY增长率而非3年CAGR — 单年波动影响较大
2. 缺少PS指标 — 对亏损公司的估值比较缺失
3. 缺少派息率/连续分红/股息增长数据
4. 纯横截面排名，无历史纵向对比
5. FCF收益率受市值波动影响
6. 板块仅31只股票，样本量较小，百分位排名区分度有限
