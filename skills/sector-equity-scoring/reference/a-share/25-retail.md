# 商贸零售板块价值评分公式 V2 (Retail Sector Value Score Formula V2)

## 概述

本公式是A股商贸零售板块（申万一级：商贸零售，东方财富板块代码BK1213）量化价值评分体系。
评分范围0-100分，分数越高表示该股票越具有投资价值。

**核心设计理念：**
1. ROE(12%)为主锚指标 — 零售行业盈利能力分化明显，高ROE体现经营效率与商业模式优势
2. 成长性权重(20%)平衡配置 — 零售行业增长主要来源于门店扩张和品类优化，营收增长与利润增长等权
3. 估值权重(20%)适中配比 — 零售行业PE分布极宽(9.6~4706)，低估值是重要筛选条件
4. 毛利率(6%)为行业特色指标 — 区分高毛利专卖零售(博士眼镜53.8%)与低毛利贸易商(五矿发展3.5%)
5. **18个非零权重指标全部参与回测，100%权重覆盖（FCF收益率已修复）**

**V1→V2修复：**
- FCF收益率计算修复：使用f20(总市值)替代f116(返回0)，FCF收益率 = FCFF_BACK / f20 × 100
- 新增"指标"列(X/18)显示每只股票有效指标数
- 现金流权重重新分配：CF/净利润 4%，FCF收益率 4%，CF/营收 3%

## 第三部分：公式

### V2评分规则

```
Value_Score = Sum(weight_i * score_i) / Sum(weight_i_participated)

score_i = industry_rank_percentile (0-100)
  越低越好: score = 100 - percentile
  越高越好: score = percentile
  percentile = (rank - 1) / (N_valid - 1) * 100
```

### 特殊处理

| 规则 | 说明 |
|------|------|
| PE < 0 | 亏损股不参与PE排名，PE权重不计入总权重 |
| ROE cap | ROE ≤ 60%，超过截断为60% |
| ROIC cap | ROIC ≤ 60%，超过截断为60% |
| 成长cap | 营收增长率、净利润增长率 cap 至 ±200% |
| 缺失值 | 缺失指标权重不参与，总分除以实际参与权重归一化 |
| 覆盖率惩罚 | 有效指标 < 10 个时，总分 *= (count / 18) |

### V2 cap应用统计

| Cap类型 | 触发次数 | 示例 |
|---------|---------|------|
| ROE ≤ 60% | 1 | 中成股份(35.2%→60.0%) |
| ROIC ≤ 60% | 0 | — |
| 净利润增长率 ±200% | 14 | 赫美集团, 宁波中百, 豫园股份等 |
| 营收增长率 ±200% | 0 | — |

### 公式简记

```
估值(20%):   PE 7%(low好)   PB 5%(low好)   股息率 8%(high好)
盈利(30%):   ROE 12%(high好) ROA 3%(high好) ROIC 5%(high好) 毛利率 6%(high好) 净利率 4%(high好)
成长(20%):   营收增 10%(high好) 利润增 10%(high好)
现金流(11%): CF/净利润 4%(high好) FCF收益率 4%(high好) CF/营收 3%(high好)

质量(8%):    毛利率变化 5%(high好) 速动比率 3%(high好)
```
