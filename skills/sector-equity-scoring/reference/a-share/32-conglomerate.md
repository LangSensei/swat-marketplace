# 综合板块价值评分公式 (Conglomerate/Diversified Sector Value Score Formula)

## 概述

本公式是A股综合板块（申万一级：综合，东方财富板块代码BK0539）量化价值评分体系。
评分范围0-100分，分数越高表示该股票越具有投资价值。

**核心设计理念：**
1. ROE(10%)为主锚指标 — 综合板块ROE分化极大（粤桂12.8% vs 亚泰-41.3%），ROE代表持续盈利能力
2. 成长性权重(16%)适度偏高 — 综合板块业务多元，成长性差异显著（天宸营收+131.7% vs 三木-51.1%），区分度高
3. 毛利率同比变化权重(7%)偏高 — 异质性板块中，毛利率趋势比绝对值更能反映经营方向
4. 股息率权重(7%)适中 — 综合板块股息率整体偏低（最高浙农3.04%），不宜过高加权
5. 风险权重(13%)标准偏高 — 板块内杠杆分化大（宁波联合17.5% vs 亚泰95.2%），D/A和带息负债率区分度高
6. 盈利能力权重(29%)标准 — ROE+ROA+ROIC+毛利率+净利率均衡分配
7. 18个非零权重指标参与回测，100%权重覆盖
8. 评分规则全套应用：增长率±200%封顶、ROE/ROIC≤60%封顶、PE<0排除、覆盖率惩罚

**行业特点：**
- 17只股票（排除2只B股后），涵盖多元化集团/贸易/投资控股等子行业
- 3只亏损股（17.6%），整体亏损比例适中
- 业务模式高度异质：粤桂（糖+造纸）、东阳光（铝+医药）、浙农（农资分销）
- PE中位数68.9（仅正PE股票），PB中位数2.30，估值整体偏高
- 3只利润增速被±200%封顶（综艺股份+261.1%→200%、天宸股份+839.5%→200%、泰达股份-350.7%→-200%）
- 0只ROE/ROIC被60%封顶
- 股息率整体偏低：最高浙农3.04%、南京公用1.00%、悦达投资0.97%

## 第三部分：公式

```
Value_Score = base_score * coverage_penalty
base_score = Sum(weight_i * score_i) / Sum(weight_i_participated)
score_i = industry_rank_percentile (0-100)
  越低越好: score = (1 - (rank - 1) / (N_valid - 1)) * 100  (rank ascending)
  越高越好: score = (1 - (rank - 1) / (N_valid - 1)) * 100  (rank descending)

Scoring rules:
  ROE cap: min(ROE, 60%)
  ROIC cap: min(ROIC, 60%)
  Growth cap: clip(growth, -200%, +200%)
  PE: loss-making (PE<0) excluded from PE ranking
  Coverage penalty: if indicators < 10, score *= (count / 18)
  Missing: excluded from that indicator, weight redistributed

Weight table:
  PE:         7% (low好)    PB:        5% (low好)    股息率:     7% (high好)
  ROE:       10% (high好)   ROA:       4% (high好)   ROIC:       5% (high好)
  毛利率:     5% (high好)   净利率:     5% (high好)
  营收增长:    8% (high好)   利润增长:   8% (high好)
  CF/净利润:   4% (high好)   FCF收益率:  4% (high好)   CF/营收:    3% (high好)
  资产负债率:  5% (low好)    流动比率:   4% (high好)   带息负债率:  4% (low好)
  毛利率变化:  7% (high好)   速动比率:   5% (high好)
```
