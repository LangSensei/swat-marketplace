# 银行板块价值评分公式 (Banking Sector Value Score Formula)

## 概述

本公式是A股银行板块（申万一级：银行）量化价值评分体系，评分范围0-100分。
分数越高表示该银行股票越具有投资价值（兼顾估值便宜度和基本面质量）。

### 标准18指标中未采用的指标

银行板块因行业特性，从标准18指标模板中移除了以下8个指标，同时新增3个银行专属指标(NPL、拨备覆盖率、净息差)，最终使用13指标、95%权重覆盖。

| 未采用指标 | 未采用原因 |
|-----------|-----------|
| ROIC | 银行资本结构以存贷为主，投入资本回报率(ROIC)框架不适用于金融机构 |
| 毛利率 | 银行无销售成本(COGS)概念，利差收入模型下毛利率无意义 |
| FCF收益率 | 银行现金流报表结构与实体企业根本不同，自由现金流(FCF)计算不适用 |
| 经营CF/净利润 | 银行经营性现金流受存贷款波动影响极大，与净利润比值不具可比性 |
| 经营CF/营收 | 同上原因，银行CF指标与非金融行业不可横向对比 |
| 流动比率 | 银行本质是期限错配业务，流动比率远低于1为常态，不具区分度 |
| 速动比率 | 同流动比率，对银行无区分意义 |
| 毛利率同比变化 | 毛利率本身不适用，变化率同样无意义 |

## 第三部分：公式

```
Value_Score = Sum(weight_i * score_i)

score_i = industry_rank_percentile (0-100)
  越低越好: score = 100 - percentile
  越高越好: score = percentile
  percentile = (rank - 1) / (N_valid - 1) * 100

NIM缺失处理: 缺失股票NIM权重置0，总分除以0.95归一化
增长率封顶: 营收增长率和净利润增长率超过±200%时，一律按±200%计算后再排名（压缩低基数暴增的极端值）
```

## 第五部分：合理性检验

### 结论

公式成功实现了以下目标：
1. **价值陷阱识别大幅改善**：贵阳银行从#2降至#21，华夏银行从#4降至#20
2. **质量溢价合理反映**：招商银行从#15升至#7，杭州银行从#25升至#12
3. **国有大行排名合理**：PE 7-8.4x偏高+NPL 1.2-1.3%中等→#30-38区间

### 局限性

1. NIM覆盖率31/42（11只仅三季报无NIM数据）
2. 使用YoY增长率而非3年CAGR
3. 缺少派息率/连续分红/股息增长数据（8%权重重分配）
4. 纯横截面排名，无历史纵向对比
