# 有色金属板块价值评分公式 (Non-ferrous Metals Sector Value Score Formula)

## 概述

本公式是A股有色金属板块（东方财富板块代码BK0478）量化价值评分体系。
评分范围0-100分，分数越高表示该股票越具有投资价值。

**核心设计理念：**
1. ROE(12%)为主锚指标 — 有色金属ROE分化大（黄金矿企20%+ vs 锂业周期低谷<5%）
2. 估值权重(20%)适度偏高 — 周期性强，低PE/PB是安全边际关键
3. 成长性权重(18%)中性 — 金价驱动黄金股高增长，锂价低迷拖累锂业
4. 现金流权重(13%)偏高 — 矿企资本开支大，现金流质量区分真实盈利能力
5. 风险类(12%)偏高 — 周期行业杠杆波动大，D/A+流动比率+带息负债三重约束
6. 质量类(8%)中毛利率变化(4%)+速动比率(4%) — 关注周期拐点信号
7. 18个非零权重指标参与回测，101%权重（归一化处理）
8. 评分规则全套应用：增长率±200%封顶、ROE/ROIC≤60%封顶、PE<0排除、覆盖率惩罚

**行业特点：**
- 139只股票（排除B股+ST/*ST），涵盖贵金属、工业金属、稀有金属、能源金属等
- 11只亏损股（7.9%），主要为锂业/小市值公司
- PE中位数30.6（黄金股PE偏高20-40，铝企PE偏低8-20）
- 18只股票利润增速被±200%封顶（12.9%），1只营收增速被封顶
- 子行业分化显著：黄金矿企整体强势，锂业周期低谷

## 第三部分：公式

```
Value_Score = base_score * coverage_penalty
base_score = Sum(weight_i * score_i) / Sum(weight_i_participated)
score_i = industry_rank_percentile (0-100)
  越低越好: score = 100 - percentile
  越高越好: score = percentile
  percentile = (rank - 1) / (N_valid - 1) * 100

Scoring rules:
  ROE cap: min(ROE, 60%)
  ROIC cap: min(ROIC, 60%)
  Growth cap: clip(growth, -200%, +200%)
  PE: loss-making (PE<0) excluded from PE ranking
  Coverage penalty: if indicators < 10, score *= (count / 18)
  Missing: excluded from that indicator, weight redistributed

Weight table:
  PE:         6% (low好)    PB:        6% (low好)    股息率:     8% (high好)
  ROE:       12% (high好)   ROA:       3% (high好)   ROIC:       5% (high好)
  毛利率:     7% (high好)   净利率:     3% (high好)
  营收增长:    8% (high好)   利润增长:  10% (high好)
  CF/净利润:   5% (high好)   FCF收益率:  4% (high好)   CF/营收:    4% (high好)
  资产负债率:  4% (low好)    流动比率:   4% (high好)   带息负债率:  4% (low好)
  毛利率变化:  4% (high好)   速动比率:   4% (high好)
```
