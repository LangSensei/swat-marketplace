# 保险板块价值评分公式 (Insurance Sector Absolute-Value Scoring Formula)

## 概述

本公式是A股保险板块（东方财富板块：BK0474）量化价值评分体系，评分范围0-100分。
分数越高表示该保险股票越具有投资价值（兼顾估值便宜度和基本面质量）。

**核心创新：绝对值区间评分（Absolute-Value Interval Scoring）**

与非银公式使用的排名分位数法不同，本公式采用绝对值区间评分。原因：
- 保险子行业仅5只A股，排名分位数法下每个排名=20个百分点，颗粒度过粗
- 绝对值区间法基于行业历史规范设定评分边界，不受样本量限制
- 每个指标设5-7个区间，评分从95（极优）到15-25（极差），依据A股保险业2020-2025年历史数据

## 第三部分：公式

```
Score = Σ(weight_i × interval_score_i) / Σ(weight_i for available indicators) × 100%

interval_score_i = 基于绝对值区间的固定分数(15-95分)
  越低越好: 值落入哪个区间，取对应分数
  越高越好: 值落入哪个区间，取对应分数

缺失处理: 缺失指标权重不参与分子分母，总分除以实际权重归一化
```

**与排名分位数法的区别：**

| 特性 | 排名分位数法 | 绝对值区间(本公式) |
|------|----------------|-------------------|
| 评分依据 | 5只股票的相对排名 | 行业历史绝对值区间 |
| 颗粒度 | 每名=20百分点 | 每区间15-20分，5-7个区间 |
| 样本敏感性 | 高（加1只股票改变全部排名） | 低（与样本量无关） |
| 跨时间可比性 | 低（排名随截面变化） | 高（绝对区间固定） |

## 第六部分：合理性检验

### 与非银公式对比

| 股票 | 排名(分位数法) | 得分 | 本公式排名(绝对值法) | 本公式得分 | 变化 |
|------|-----------------|--------|---------------------|-----------|------|
| 新华保险 | #1 | 55.7 | #1 | 69.3 | 不变，分数↑ |
| 中国太保 | #2 | 55.3 | #3 | 61.8 | ↓1位 |
| 中国人寿 | #3 | 50.8 | #2 | 67.7 | ↑1位 |
| 中国人保 | #4 | 43.9 | #4 | 60.3 | 不变，分数↑ |
| 中国平安 | #5 | 43.7 | #5 | 59.0 | 不变，分数↑ |

### 排名变化分析

1. **新华保险稳居第一**：PE 5.49最低 + ROE 34.7%最高 + 股息4.27% + 高增长 → 估值便宜+基本面强劲的"双击"标的
2. **中国人寿升至第二**（从分位数法的#3）：NBV增长率35%最高 + 利润增速44%最快 + 退保率0.95%最低 → 绝对值法更好地捕捉了其成长优势。分位数法中因5股排名，人寿NBV的#1仅比#2多20百分点，绝对值法下35%直接获95分(vs太保19.8%获65分)
3. **中国太保降至第三**（从分位数法的#2）：偿付能力273%最高(95分) + PB 1.21中等(50分) → 均衡型，但无极端优势。"安全垫最厚"但"不是最便宜也不是增长最快"
4. **中国人保稳居第四**：ROA 3.32%最高(含产险) + 低负债率79.3% → 但NBV缺失(权重10%不参与)、利润增速仅8.8%拖累
5. **中国平安稳居第五**：PB 1.06最低 + 股息4.61%最高 → 但ROE 14%行业最低 + 营收增长2.1%最慢 + 利润增速6.5%最低 → 估值便宜但增长乏力。**注意：平安的绝对估值仍合理(PE 7.86/PB 1.06/股息4.61%)，但在5家中相对弱势**

### 绝对值法 vs 分位数法的效果对比

| 方面 | 分位数法 | 绝对值法(本公式) |
|------|-------------|-----------------|
| 分数范围 | 43.7-55.7 (12分差) | 59.0-69.3 (10.3分差) |
| 区分度 | #1和#2仅差0.4分 | #1和#2差1.6分 |
| NBV贡献 | 人寿#1=100, 太保#2=75 (差25) | 人寿35%=95分, 太保19.8%=65分 (差30) |
| 对极端值敏感度 | 被排名压缩 | 直接反映绝对差距 |

### 结论

绝对值区间评分法在5股小样本场景下优于排名分位数法：
1. 更好地反映了指标的绝对差距（如NBV 35% vs 19.8%不再被压缩为排名差1位=20百分点）
2. 跨时间可比：如果某公司PE从8降到5，绝对值法立即反映(55→85)，分位数法可能不变(如果排名不变)
3. 排名大方向与分位数法一致（新华#1、平安#5），但中间排序更精确

### 局限性

1. 区间边界基于2020-2025年历史数据设定，若行业估值中枢系统性迁移需重新校准
2. ROE受IFRS17新准则影响，2024-2025年部分公司ROE偏高(如新华34.7%)，可能是一过性
3. NBV增长率和综合投资收益率各有1只股票缺失(覆盖率80%)
4. 有息负债率因中国平安含银行子公司而无法使用
5. 使用YoY增长率而非3年CAGR，受单年波动影响大
