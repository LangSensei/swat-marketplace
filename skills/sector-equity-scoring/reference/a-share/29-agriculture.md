# 农林牧渔板块价值评分公式 (Agriculture Sector Value Score Formula)

## 概述

本公式是A股农林牧渔板块（申万一级农林牧渔，东方财富板块代码BK0433）量化价值评分体系。
0-100分，分数越高表示该股票越具有投资价值。

**核心设计理念：**
1. ROE(12%)为主锚指标 — 农林牧渔ROE分化极大（中水渔业21.1%、牧原股份20.6% vs 獐子岛-91.2%），高ROE代表持续竞争优势
2. 20%(估值权重 PE 6%+PB 6%+股息率 8%；猪周期导致PE波动极大，PB对重资产养殖企业更有参考价值)标准 
3. 成长性权重(15%)适度 — 猪周期恢复/崩溃导致利润增速剧烈波动，12只利润增速被±200%封顶（13.2%），增长率封顶至关重要
4. 现金流权重(13%)偏高 — 养殖业资本密集，FCF收益率(5%)+CF/净利润(5%)权重高于标准模板
5. 风险类(12%)标准 — D/A 5%+流动比率4%+带息负债率3%，养殖企业杠杆水平关键
6. 质量类(10%)中毛利率变化(5%)5%(+速动比率 周期行业中毛利率稳定性反映定价权) 
7. 18个非零权重指标参与回测，100%权重覆盖
8. 评分规则全套应用：增长率±200%封顶、ROE/ROIC≤60%封顶、PE<0排除、覆盖率惩罚

**行业特点：**
- 91只股票（排除6只ST+3B股），涵盖畜牧养殖/种业/水产/饲料/动保/食品加工等子行业
- 22只亏损股（24.2%），猪周期底部/下行期亏损比例显著
- PE中位数偏高（正PE股票中位数39.1），PB中位数2.54
- 12只利润增速被±200%封顶（13.2%），2只营收增速被±200%封顶
- 0只ROE/ROIC被60%封顶
- 子行业分化显著：动保(科前生物、普莱柯)高毛利高ROE vs 养殖(新五丰、金新农)周期性强亏损

## 第三部分：公式

```
Value_Score = base_score * coverage_penalty
base_score = Sum(weight_i * score_i) / Sum(weight_i_participated)
score_i = industry_rank_percentile (0-100)
  越低越好: score = (N - rank_asc) / (N - 1) * 100
  越高越好: score = (N - rank_desc) / (N - 1) * 100

Scoring rules:
  ROE cap: min(ROE, 60%)
  ROIC cap: min(ROIC, 60%)
  Growth cap: clip(growth, -200%, +200%)
  PE: loss-making (PE<0) excluded from PE ranking
  Coverage penalty: if indicators < 10, score *= (count / 18)
  Missing: excluded from that indicator, weight redistributed

Weight table:
  PE:         6% (low好)    PB:        6% (low好)    股息率:     8% (high好)
  ROE:       12% (high好)   ROA:       3% (high好)   ROIC:       5% (high好)
  毛利率:     5% (high好)   净利率:     5% (high好)
  营收增长:    7% (high好)   利润增长:   8% (high好)
  CF/净利润:   5% (high好)   FCF收益率:  5% (high好)   CF/营收:    3% (high好)
  资产负债率:  5% (low好)    流动比率:   4% (high好)   带息负债率:  3% (low好)
  毛利率变化:  5% (high好)   速动比率:   5% (high好)
```
