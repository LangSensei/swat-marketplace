# 国防军工板块价值评分公式

## 概述

本公式是A股国防军工板块（申万一级：国防军工，东方财富板块代码BK1204）量化价值评分体系。
评分范围0-100分，分数越高表示该股票越具有投资价值。

**核心设计理念：**
1. ROE(12%)为主锚指标 — 军工企业ROE普遍偏低(中位数4-8%)，高ROE是显著优势
2. 毛利率(5%)区分子行业 — 军工电子/新材料50%+ vs 船舶/整机10-15%
3. 成长性权重(15%)适度降低 — 军工订单批量性强、周期性波动大，降低增长权重避免失真
4. 估值权重(20%)标准 — 军工PE普遍偏高(中位数50x+)，PE仍有区分度
5. 现金流权重(13%)偏高 — 军工预付款/应收款周期长，现金流质量是重要区分维度
6. 18个非零权重指标参与回测，100%权重覆盖
7. 评分规则全套应用：增长率±200%封顶、ROE/ROIC≤60%封顶、PE<0排除、覆盖率惩罚

**行业特点：**
- 143只股票（含军工电子、航空航天、船舶、兵器等子行业）
- 44只亏损股（30.8%），高于家电(11%)和食品饮料(13%)，与计算机(44.3%)接近
- SOE(国有企业)主导，ROE普遍较低，中位数4-8%
- 军工订单批量性强，营收/利润波动大 → 25只股票利润增速被±200%封顶
- 毛利率分化显著：军工电子/新材料50-70% vs 船舶/整机10-20%
- 行业PE普遍偏高(中位数50x+)，传统估值框架需适应

## 第三部分：公式

```
Value_Score = base_score * coverage_penalty
base_score = Sum(weight_i * score_i) / Sum(weight_i_participated)
score_i = industry_rank_percentile (0-100)
  lower_better: score = 100 - percentile
  higher_better: score = percentile
  percentile = (rank - 1) / (N_valid - 1) * 100

Scoring rules:
  ROE cap: min(ROE, 60%)
  ROIC cap: min(ROIC, 60%)
  Growth cap: clip(growth, -200%, +200%)
  PE: loss-making (PE<0) excluded from PE ranking
  Coverage penalty: if indicators < 10, score *= (count / 18)
  Missing: excluded from that indicator, weight redistributed
```
