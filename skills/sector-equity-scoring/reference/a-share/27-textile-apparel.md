# 纺织服饰板块价值评分公式 V2 (Textile & Apparel Sector Value Score Formula V2)

## 概述

本公式是A股纺织服饰板块（申万一级：纺织服饰，东方财富板块代码BK0436）量化价值评分体系。
评分范围0-100分，分数越高表示该股票越具有投资价值。

**核心设计理念：**
1. ROE(13%)为主锚指标 — 纺织服饰品牌企业依靠品牌溢价维持高ROE（海澜之家12%+，华利集团19%+）
2. 毛利率(6%)为行业特色指标 — 区分高毛利品牌（比音勒芬75%）与低毛利OEM制造商（百隆东方14%），品牌力直接体现
3. 成长性权重(18%)适中配置 — 服饰行业有周期性，但品牌扩张和品类创新仍是增长核心驱动力
4. 估值权重(20%)强调股息率(8%) — 成熟纺织企业派息稳定（水星家纺4.3%、海澜之家6.1%）
5. 风险权重(12%)略高 — OEM制造商资产偏重，负债结构差异大，需关注偿债能力
6. **18个非零权重指标全部参与回测，100%权重覆盖**

## 第三部分：公式

### V2评分规则

```
Value_Score = Sum(weight_i * score_i) / Sum(weight_i_participated)

score_i = industry_rank_percentile (0-100)
  越低越好: score = 100 - percentile
  越高越好: score = percentile
  percentile = (rank - 1) / (N_valid - 1) * 100
```

### 特殊处理

| 规则 | 说明 |
|------|------|
| PE < 0 | 亏损股不参与PE排名，PE权重不计入总权重 |
| ROE cap | ROE ≤ 60%，超过截断为60% |
| ROIC cap | ROIC ≤ 60%，超过截断为60% |
| 成长cap | 营收增长率、净利润增长率 cap 至 ±200% |
| 缺失值 | 缺失指标权重不参与，总分除以实际参与权重归一化 |
| 覆盖率惩罚 | 有效指标 < 10 个时，总分 *= (count / 18) |

### V2 cap应用统计

| Cap类型 | 触发次数 | 示例 |
|---------|---------|------|
| ROE ≤ 60% | 0 | — |
| ROIC ≤ 60% | 0 | — |
| 净利润增长率 ±200% | 14 | 朗姿股份(366.9%), 红豆股份(-6741.6%), 华纺股份(-3223.8%), 歌力思(427.3%), 金春股份(484.7%), 明牌珠宝(-1502.1%)等 |
| 营收增长率 ±200% | 0 | — |

### 公式简记

```
估值(20%):   PE 7%(low好)   PB 5%(low好)   股息率 8%(high好)
盈利(30%):   ROE 13%(high好) ROA 3%(high好) ROIC 5%(high好) 毛利率 6%(high好) 净利率 3%(high好)

```
