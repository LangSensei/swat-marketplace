# 医药生物板块价值评分公式 (Pharmaceutical & Biotech Sector Value Score Formula)

## 概述

本公式是A股医药生物板块（申万一级：医药生物，东方财富板块代码BK1216）量化价值评分体系。
评分范围0-100分，分数越高表示该股票越具有投资价值。

**核心设计理念：**
1. 成长性权重最高(25%) — 医药是典型成长型行业，创新药/器械增长是核心驱动力
2. ROE(12%)为盈利主锚 — 筛选持续盈利能力强的优质公司
3. 毛利率(7%)为行业质量指标 — 创新药80%+ vs 仿制药30-50%，区分创新实力
4. 估值权重仅18% — 医药板块121/506亏损，PE区分力有限
5. R&D费用不可用 — API不提供研发费用数据，通过成长性间接反映研发成效
6. 18个非零权重指标参与回测，100%权重覆盖

## 第三部分：公式

```
Value_Score = Sum(weight_i * score_i) / Sum(weight_i_participated) * 1.0

score_i = industry_rank_percentile (0-100)
  越低越好: score = 100 - percentile
  越高越好: score = percentile
  percentile = (rank - 1) / (N_valid - 1) * 100

PE处理: 亏损股(PE<0)PE指标得分为0，其余指标正常参与
增长率封顶: 营收增长率和净利润增长率超过±200%时，一律按±200%计算后再排名
  原因: 低基数暴增(如通化东宝利润增长1911%)会扭曲排名，封顶压缩极端值
  示例: 通化东宝利润增长1911% → cap至200%后参与排名，仍属高增长但不再异常突出
缺失值处理: 缺失指标权重不参与，总分除以实际参与权重归一化
```

### 局限性

1. 使用YoY增长率而非3年CAGR — 单年波动影响较大（如通化东宝利润增长1911%因低基数效应）
2. 缺少R&D/营收指标 — 无法直接量化研发投入强度
3. 缺少PS指标 — 对亏损公司的估值比较缺失
4. 缺少管线价值评估 — 创新药管线是核心资产但无法量化
5. 缺少派息率/连续分红/股息增长数据
6. 纯横截面排名，无历史纵向对比
7. 部分新上市公司财务数据仅一期，增长率可能失真
