# 电力设备板块价值评分公式 (Power Equipment Sector Value Score Formula)

## 概述

本公式是A股电力设备板块（申万一级：电力设备，东方财富板块代码BK1200）量化价值评分体系。
评分范围0-100分，分数越高表示该股票越具有投资价值。

**核心设计理念：**
1. ROE(12%)为主锚指标 — 电网龙头维持10-20% ROE，电池/光伏波动大
2. 估值权重(20%)高于电子(18%) — 电网设备PE更稳定，估值锚定更可靠
3. 成长性权重(22%)低于电子(25%) — 电网设备稳增长，不需过高成长权重
4. 现金流权重(12%)高于电子(10%) — 重资产行业，现金流质量更关键
5. 毛利率(7%)为行业特色指标 — 光伏设备毛利率波动剧烈(从50%+跌至负值)，电网设备相对稳定
6. 18个非零权重指标参与回测，100%权重覆盖
7. 评分规则全套应用：增长率±200%封顶、ROE/ROIC≤60%封顶、PE<0排除、覆盖率惩罚

**行业特点：**
- 402只股票，涵盖电网设备、电池、光伏设备、风电设备、电机、其他电源设备六大子行业
- 99只亏损股（24.6%），主要集中在光伏和电池子行业（产能过剩周期）
- PE中位数57.5（无需/100转换），PB中位数3.6
- 49只股票利润增速被±200%封顶（12.2%），2只营收增速被封顶
- 子行业分化显著：电网设备（高壁垒、稳增长）vs 光伏（强周期、高波动）

## 第三部分：公式

```
Value_Score = base_score * coverage_penalty
base_score = Sum(weight_i * score_i) / Sum(weight_i_participated)
score_i = industry_rank_percentile (0-100)
  越低越好: score = 100 - percentile
  越高越好: score = percentile
  percentile = (rank - 1) / (N_valid - 1) * 100

Scoring rules:
  ROE cap: min(ROE, 60%)
  ROIC cap: min(ROIC, 60%)
  Growth cap: clip(growth, -200%, +200%)
  PE: loss-making (PE<0) excluded from PE ranking
  Coverage penalty: if indicators < 10, score *= (count / 18)
  Missing: excluded from that indicator, weight redistributed

Weight table:
  PE:         7% (low好)    PB:        5% (low好)    股息率:     8% (high好)
  ROE:       12% (high好)   ROA:       3% (high好)   ROIC:       5% (high好)
  毛利率:     7% (high好)   净利率:     3% (high好)
  营收增长:   10% (high好)   利润增长:  12% (high好)
  CF/净利润:   5% (high好)   FCF收益率:  4% (high好)   CF/营收:    3% (high好)
  资产负债率:  4% (low好)    流动比率:   3% (high好)   带息负债率:  3% (low好)
  毛利率变化:  3% (high好)   速动比率:   3% (high好)
```
