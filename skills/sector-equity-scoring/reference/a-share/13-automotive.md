# 汽车板块价值评分公式 (Automotive Sector Value Score Formula)

## 概述

本公式是A股汽车板块（申万一级：汽车，东方财富板块代码BK0481）量化价值评分体系。
评分范围0-100分，分数越高表示该股票越具有投资价值。

**核心设计理念：**
1. ROE(12%)为主锚指标 — 汽车零部件ROE 10-28% vs 整车OEM 5-15%，区分度大
2. 成长性权重(22%)与传媒持平 — EV转型带来高弹性增长
3. 盈利能力权重(28%)与传媒持平 — 零部件高毛利(30-50%) vs 整车低毛利(8-20%)
4. 行业质量权重(10%)高于家电(8%) — 毛利率变化反映EV渗透率提升
5. 18个非零权重指标参与回测，100%权重覆盖
6. 无需子行业拆分 — 零部件(高ROE/高毛利)与整车(低PE/高营收)自然区分

**行业特点：**
- 268只股票（去重后），为申万一级中大型板块
- 33只亏损股（12.3%），低于计算机(44.3%)和传媒(33.1%)
- 28只股票利润增速被±200%封顶，反映EV转型期盈利波动
- 零部件供应商占比高(约60%)，比亚迪归入电力设备
- EV转型创造赢家(智能座舱/三电/轻量化)和输家(传统发动机/变速箱)

## 第三部分：公式

```
Value_Score = Sum(weight_i * score_i) / Sum(weight_i_participated)

score_i = industry_rank_percentile (0-100)
  lower is better: score = 100 - percentile
  higher is better: score = percentile
  percentile = (rank - 1) / (N_valid - 1) * 100

V2 rules:
  Growth cap: clip(growth, -200%, +200%)
  ROE/ROIC cap: clip(value, upper=60%)
  PE: loss-making (PE<0) excluded from PE ranking
  Coverage penalty: if indicators < 10, score *= (count / 18)
  Missing: excluded from that indicator, weight redistributed
```
