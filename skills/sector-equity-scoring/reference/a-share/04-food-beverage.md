# 食品饮料板块价值评分公式 (Food & Beverage Sector Value Score Formula)

## 概述

本公式是A股食品饮料板块（申万一级：食品饮料，东方财富板块代码BK0438）量化价值评分体系。
评分范围0-100分，分数越高表示该股票越具有投资价值。

**核心设计理念：**
1. ROE(15%)为主锚指标 — 食品饮料龙头企业持续维持20%+ROE
2. 毛利率(8%)为行业特色指标 — 白酒企业80%+毛利率是品牌护城河的直接体现
3. 成长性权重(20%)高于银行公式(10%) — 消费品增长更重要
4. 估值权重(20%)低于银行公式(35%) — 消费品合理溢价更大
5. 18个非零权重指标参与回测，100%权重覆盖

## 第三部分：公式

```
Value_Score = Sum(weight_i * score_i) / Sum(weight_i_participated)

score_i = industry_rank_percentile (0-100)
  越低越好: score = 100 - percentile
  越高越好: score = percentile
  percentile = (rank - 1) / (N_valid - 1) * 100

PE处理: 亏损股(PE<0)不参与PE排名
增长率封顶: 营收增长率和净利润增长率超过±200%时，一律按±200%计算后再排名（压缩低基数暴增的极端值）
缺失值处理: 缺失指标权重不参与，总分除以实际参与权重归一化
```
