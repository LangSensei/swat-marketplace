# 非银金融板块价值评分公式 (Non-bank Financials Value Score Formula)

## 概述

本公式是A股非银金融板块（申万一级：非银金融）量化价值评分体系，评分范围0-100分。
覆盖保险（BK0474, 5只）和证券（BK0473, 49只）两个子行业，采用条件权重公式（Conditional-Weight Formula）。

## 第三部分：公式

```
Score(stock) = Σ (weight_i × normalized_score_i) / Σ (weight_i for available indicators)

normalized_score_i = industry_rank_percentile (0-100)
  越低越好: score = 100 - percentile
  越高越好: score = percentile
  percentile = (rank - 1) / (N_valid - 1) * 100

排名池: 保险(5只)和证券(49只)分开排名
增长率封顶: 营收增长率和净利润增长率超过±200%时，一律按±200%计算后再排名（压缩低基数暴增的极端值）
缺失处理: 缺失指标权重不参与，总分除以实际参与权重归一化
PE异常处理: PE<0(亏损)或PE>200 → 该指标设为缺失
```

### 结论

公式的核心优势：
1. **从2指标简化回测升级为14指标全量回测**，权重覆盖从~30%提升至~97%
2. **保险专属指标成功获取**：NBV增长率(80%)、综合投资收益率(80%)、偿付能力(100%)
3. **券商专属指标首次纳入**：资本杠杆率(86%)、风险覆盖率(86%)，显著增强风控维度区分
4. **排名变化符合逻辑**：新华保险凭超低PE+超高ROE登顶保险，国泰海通凭合并后业绩爆发稳居券商第一

### 局限性

1. 保险仅5只股票，排名分位数颗粒度有限（每名差20个百分点）
2. P/EV无法从API获取，改用PB替代（相关性高但非完全替代）
3. 使用YoY增长率而非3年CAGR
4. 部分券商缺少股息率(6只)和资本杠杆率/风险覆盖率(7只)数据
5. 保险与券商分池排名，跨子行业直接比较分数需谨慎
6. 纯横截面排名，无历史纵向对比
