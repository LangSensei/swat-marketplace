# 传媒板块价值评分公式 (Media Sector Value Score Formula)

## 概述

本公式是A股传媒板块（申万一级：传媒，东方财富板块代码BK0486）量化价值评分体系。
评分范围0-100分，分数越高表示该股票越具有投资价值。

**核心设计理念：**
1. ROE(12%)为主锚指标 — 游戏公司ROE 20-33% vs 出版5-12%，区分度极大
2. 股息率(8%)权重为全系列最高 — 出版板块股息率4-7%是核心吸引力
3. 成长性权重(22%)高于食品饮料(20%)但低于电子(25%) — 游戏/影视增长弹性大但不可靠
4. 现金流权重(12%)与通信持平 — 传媒行业利润含金量差异大（游戏强、影视弱）
5. 估值权重(20%)与通信持平 — 影视PE波动极大，但PE仍是区分高低估的核心
6. 18个非零权重指标参与回测，100%权重覆盖
7. 无需子行业拆分 — 18指标体系自然区分游戏（高毛利/高CF）、出版（高股息/稳增长）、影视（高波动）

**行业特点：**
- 130只股票，与通信板块规模相当
- 43只亏损股（33.1%），为全系列最高（电子22.5%、通信28.5%）
- 游戏公司毛利率50-94%、出版27-48%、影视14-73%（差异极大）
- 2015-2018年M&A潮遗留大量商誉减值公司（华谊兄弟、博纳影业等）
- 21只股票利润增速被±200%封顶（16.2%），仅次于电子(11.3%)
- 经营CF/净利润覆盖率仅66.2%（全系列最低），因亏损股占比高

## 第三部分：公式

```
Value_Score = Sum(weight_i * score_i) / Sum(weight_i_participated)

score_i = industry_rank_percentile (0-100)
  越低越好的指标: score = 100 - percentile
  越高越好的指标: score = percentile
  percentile = (rank - 1) / (N_valid - 1) * 100

评分规则:
  成长指标cap: 营收增长率/净利润增长率 clip至[-200%, +200%]
  ROE/ROIC cap: clip至≤60%
  PE处理: 亏损股(PE<0)不参与PE排名
  覆盖惩罚: 可用指标数<10时，得分×(指标数/18)
  缺失值处理: 缺失指标权重不参与，总分除以实际参与权重归一化
```
