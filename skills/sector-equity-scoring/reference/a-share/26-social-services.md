# 社会服务板块价值评分公式 (Social Services Sector Value Score Formula)

## 概述

本公式是A股社会服务板块（申万一级：社会服务，东方财富板块代码BK1214）量化价值评分体系。
评分范围0-100分，分数越高表示该股票越具有投资价值。

**核心设计理念：**
1. 毛利率(6%)+净利率(4%)为行业特色指标 — 社会服务行业毛利差异极大：检测行业(华测检测48.6%)、教育(行动教育78.9%)高毛利 vs 餐饮(同庆楼17.6%)、人力资源(北京人力5.5%)低毛利
2. ROE(10%)为盈利核心锚 — 区分高效率运营(行动教育31.0%)与低效率企业
3. 成长权重(18%)体现复苏弹性 — 旅游/酒店后疫情复苏周期，增长指标区分力强
4. 股息率(8%)筛选现金回报 — 区分成熟稳定标的(开普检测4.25%、北京人力4.29%)
5. 质量权重(11%)含毛利率同比变化(6%) — 反映行业竞争力变化趋势
6. **18个非零权重指标全部参与回测，100%权重覆盖（FCF收益率使用f20总市值）**

## 第三部分：公式

### 评分规则

```
Value_Score = Sum(weight_i * score_i) / Sum(weight_i_participated)

score_i = industry_rank_percentile (0-100)
  越低越好: score = 100 - percentile
  越高越好: score = percentile
  percentile = (rank - 1) / (N_valid - 1) * 100
```

### 特殊处理

| 规则 | 说明 |
|------|------|
| PE < 0 | 亏损股不参与PE排名，PE权重不计入总权重 |
| ROE cap | ROE ≤ 60%，超过截断为60% |
| ROIC cap | ROIC ≤ 60%，超过截断为60% |
| 成长cap | 营收增长率、净利润增长率 cap 至 ±200% |
| 缺失值 | 缺失指标权重不参与，总分除以实际参与权重归一化 |
| 覆盖率惩罚 | 有效指标 < 10 个时，总分 *= (count / 18) |

### 封顶规则应用统计

| Cap类型 | 触发次数 | 示例 |
|---------|---------|------|
| ROE ≤ 60% | 1 | 昂立教育(62.8%→60.0%) |
| ROIC ≤ 60% | 0 | — |
| 净利润增长率 ±200% | 2 | 电科院(-247.4%→-200%), 云南旅游(-1073.0%→-200%) |
| 营收增长率 ±200% | 0 | — |

### 公式简记

```
估值(20%):   PE 7%(low好)   PB 5%(low好)   股息率 8%(high好)
盈利(28%):   ROE 10%(high好) ROA 3%(high好) ROIC 5%(high好) 毛利率 6%(high好) 净利率 4%(high好)
成长(18%):   营收增 9%(high好) 利润增 9%(high好)
现金流(12%): CF/净利润 4%(high好) FCF收益率 4%(high好) CF/营收 4%(high好)
风险(11%):   资产负债率 4%(low好) 流动比率 3%(high好) 带息负债率 4%(low好)
质量(11%):   毛利率变化 6%(high好) 速动比率 5%(high好)
```
