# 钢铁板块价值评分公式 (Steel Sector Value Score Formula)

## 概述

本公式是A股钢铁板块（申万一级钢铁，东方财富板块代码BK0479）量化价值评分体系。
评分范围0-100分，分数越高表示该股票越具有投资价值。

**核心设计理念：**
1. ROE(12%)为主锚指标 — 钢铁行业ROE分化极大（久立特材19.9% vs 安阳钢铁-76.4%），高ROE代表持续竞争优势
2. 估值权重(20%)标准 — PE 6%+PB 6%+股息率 8%；周期行业PE波动大，PB对重资产行业更有参考价值
3. 成长性权重(15%)偏低 — 成熟周期行业，有机增长有限，6只利润增速被±200%封顶
4. 现金流权重(13%)偏高 — 资本密集型行业，FCF收益率(5%)权重高于标准模板
5. 风险类(12%)较高 — D/A 5%+流动比率4%+带息负债率3%，钢铁行业杠杆水平关键
6. 质量类(10%)中毛利率变化(5%)+速动比率(5%) — 周期行业中毛利率稳定性关键
7. 18个非零权重指标参与回测，100%权重覆盖
8. V2规则全套应用：增长率±200%封顶、ROE/ROIC≤60%封顶、PE<0排除、覆盖率惩罚

**行业特点：**
- 43只股票（排除2只B股+1只ST），涵盖特钢/普钢/钢管/矿业/铁合金等
- 10只亏损股（23.3%），行业周期底部亏损比例显著
- PE中位数19.2（仅正PE股票），PB中位数1.4
- 6只股票利润增速被±200%封顶（14.0%），0只营收增速被封顶
- 0只ROE/ROIC被60%封顶
- 子行业分化显著：特钢(久立、中信、常宝)高盈利 vs 普钢(鞍钢、山东钢铁)低盈利/亏损

## 第三部分：公式

```
Value_Score = base_score * coverage_penalty
base_score = Sum(weight_i * score_i) / Sum(weight_i_participated)
score_i = industry_rank_percentile (0-100)
  越低越好: score = 100 - percentile
  越高越好: score = percentile
  percentile = (rank - 1) / (N_valid - 1) * 100

V2 rules:
  ROE cap: min(ROE, 60%)
  ROIC cap: min(ROIC, 60%)
  Growth cap: clip(growth, -200%, +200%)
  PE: loss-making (PE<0) excluded from PE ranking
  Coverage penalty: if indicators < 10, score *= (count / 18)
  Missing: excluded from that indicator, weight redistributed

Weight table:
  PE:         6% (low好)    PB:        6% (low好)    股息率:     8% (high好)
  ROE:       12% (high好)   ROA:       3% (high好)   ROIC:       5% (high好)
  毛利率:     7% (high好)   净利率:     3% (high好)
  营收增长:    6% (high好)   利润增长:   9% (high好)
  CF/净利润:   5% (high好)   FCF收益率:  5% (high好)   CF/营收:    3% (high好)
  资产负债率:  5% (low好)    流动比率:   4% (high好)   带息负债率:  3% (low好)
  毛利率变化:  5% (high好)   速动比率:   5% (high好)
```
