# 计算机板块价值评分公式 (Computer Sector Value Score Formula)

## 概述

本公式是A股计算机板块（申万一级：计算机，东方财富板块代码BK1207）量化价值评分体系。
评分范围0-100分，分数越高表示该股票越具有投资价值。

**核心设计理念：**
1. 成长性权重最高(25%) — 软件/计算机是典型成长型行业，增长是核心驱动力
2. ROE(12%)为盈利主锚 — 筛选持续盈利能力强的优质公司
3. 毛利率(7%)为行业质量指标 — 软件公司80%+毛利率 vs 硬件30-50%，区分商业模式
4. 估值权重仅18% — 板块160/361亏损(44.3%)，PE区分力有限
5. 资产轻型特征 — 低负债率、高速动比率体现软件公司特性
6. 18个非零权重指标参与回测，100%权重覆盖
7. 评分规则全套应用：增长±200%封顶、ROE/ROIC≤60%封顶、PE<0排除、覆盖率惩罚

## 第三部分：公式

```
Value_Score = base_score * coverage_penalty
base_score = Sum(weight_i * score_i) / Sum(weight_i_participated)
score_i = industry_rank_percentile (0-100)
  lower_better: score = 100 - percentile
  higher_better: score = percentile
  percentile = (rank - 1) / (N_valid - 1) * 100

Scoring rules:
  ROE cap: min(ROE, 60%)
  ROIC cap: min(ROIC, 60%)
  Growth cap: clip(growth, -200%, +200%)
  PE: loss-making (PE<0) PE score = 0, other indicators normal
  Coverage penalty: if indicators < 10, score *= (count / 18)
  Missing: excluded from ranking, weight redistributed
```
