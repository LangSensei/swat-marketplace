# 基础化工板块价值评分公式 (Basic Chemicals Sector Value Score Formula)

## 概述

本公式是A股基础化工板块（东方财富板块代码BK1206）量化价值评分体系。
评分范围0-100分，分数越高表示该股票越具有投资价值。

**核心设计理念：**
1. ROE(12%)为主锚指标 — 化工行业ROE分化大（盐湖22%、万华14%、亏损股-30%+），高ROE代表持续竞争优势
2. 估值权重(20%)标准偏高 — 周期性行业PE波动大，高股息率龙头(国光7.6%、芭田5.3%)有突出安全边际
3. 成长性权重(18%)适中 — 高度周期性行业，增长率波动极大（40只利润增速被±200%封顶），降低权重避免过拟合
4. 现金流权重(12%)偏高 — 资本密集型行业，现金流质量区分真盈利与纸面利润
5. 毛利率(7%)为行业特色指标 — 区分特种化学品(高毛利40%+)和大宗商品化学品(低毛利15-20%)
6. 风险类(10%)均衡分配 — D/A 4%、流动比率3%、带息负债率3%，不过度惩罚合理杠杆
7. 质量类(10%)中毛利率变化(5%)+速动比率(5%) — 周期行业中毛利率稳定性尤为重要
8. 18个非零权重指标参与回测，100%权重覆盖
9. 评分规则全套应用：增长率±200%封顶、ROE/ROIC≤60%封顶、PE<0排除、覆盖率惩罚

**行业特点：**
- 402只股票（排除36只B股+10只ST/*ST），涵盖化肥/农药/氟化工/煤化工/盐化工/新材料/精细化学品等
- 79只亏损股（19.6%），周期底部亏损比例显著
- PE中位数44.1（仅正PE股票），PB中位数2.55
- 40只股票利润增速被±200%封顶（10.0%），0只营收增速被封顶
- 1只ROIC被60%封顶（正丹股份）
- 子行业分化显著：盐湖/藏格(资源型高毛利) vs 基础化工品(低毛利高周期)

## 第三部分：公式

```
Value_Score = base_score * coverage_penalty
base_score = Sum(weight_i * score_i) / Sum(weight_i_participated)
score_i = industry_rank_percentile (0-100)
  越低越好: score = 100 - percentile
  越高越好: score = percentile
  percentile = (rank - 1) / (N_valid - 1) * 100

Scoring rules:
  ROE cap: min(ROE, 60%)
  ROIC cap: min(ROIC, 60%)
  Growth cap: clip(growth, -200%, +200%)
  PE: loss-making (PE<0) excluded from PE ranking
  Coverage penalty: if indicators < 10, score *= (count / 18)
  Missing: excluded from that indicator, weight redistributed

Weight table:
  PE:         7% (low好)    PB:        5% (low好)    股息率:     8% (high好)
  ROE:       12% (high好)   ROA:       3% (high好)   ROIC:       5% (high好)
  毛利率:     7% (high好)   净利率:     3% (high好)
  营收增长:    8% (high好)   利润增长:  10% (high好)
  CF/净利润:   5% (high好)   FCF收益率:  4% (high好)   CF/营收:    3% (high好)
  资产负债率:  4% (low好)    流动比率:   3% (high好)   带息负债率:  3% (low好)
  毛利率变化:  5% (high好)   速动比率:   5% (high好)
```
