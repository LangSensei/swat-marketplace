# 轻工制造板块价值评分公式

## 概述

本文档定义轻工制造（申万一级）板块的价值评分公式，采用 百分位排名方法论，涵盖 6 大维度 18 项指标。板块包含家具定制、家居用品、建筑陶瓷、遮阳材料等细分领域，共 76 只股票（排除 ST/*ST 后 73 只参与评分）。

**数据来源：**
- 批量行情：`push2.eastmoney.com/api/qt/clist/get`（PE、PB、股息率、总市值）
- 财务数据：`datacenter-web.eastmoney.com/api/data/v1/get`（RPT_F10_FINANCE_MAINFINADATA，ROE、ROIC、毛利率、净利率、营收增长、利润增长、经营现金流、资产负债率、带息负债比、流动比率、速动比率等）
- FCF 收益率 = 自由现金流 / 总市值（f20字段）

**评分规则：**
- 排除 ST/*ST 和 B 股
- 成长指标封顶 ±200%
- ROE/ROIC 封顶 60%
- 亏损股 PE 不参与百分位排名（PE 显示"亏损"）
- 数据缺失惩罚：实际指标数/18 × 原始得分

## 第三部分：公式

### 1. 数据获取

```
# 批量行情 API
GET push2.eastmoney.com/api/qt/clist/get
  ?fs=b:BK0440
  &fields=f12,f14,f9,f23,f20,f133
  # f12=代码, f14=名称, f9=PE(TTM), f23=PB, f20=总市值, f133=股息率

# 财务数据 API
GET datacenter-web.eastmoney.com/api/data/v1/get
  ?reportName=RPT_F10_FINANCE_MAINFINADATA
  &filter=(SECURITY_CODE="{code}")
  &columns=ROEJQ,XSMLL,XSJLL,TOTALOPERATEREVETZ,PARENTNETPROFITTZ,...
```

### 2. 预处理

```python
# 排除 ST/*ST 和 B 股
stocks = [s for s in raw_stocks if not is_st(s) and not is_b_share(s)]

# 封顶处理
roe_capped    = min(roe, 60)
roic_capped   = min(roic, 60)
rev_growth    = max(-200, min(200, rev_growth_raw))
profit_growth = max(-200, min(200, profit_growth_raw))
```

### 3. 百分位排名

对每个指标，在所有有效值中计算百分位排名（0-100）：

```python
def percentile_rank(values, idx, higher_is_better=True):
    valid = sorted([v for v in values if v is not None])
    rank = valid.index(values[idx]) / (len(valid) - 1) * 100
    return rank if higher_is_better else (100 - rank)
```

- **高好指标**：百分位越高 → 得分越高
- **低好指标**：百分位越高 → 得分越低（取 100 − 百分位）
- **亏损股 PE**：不参与 PE 百分位排名，PE 得分 = 0

### 4. 加权汇总

```python
score = sum(percentile_rank[i] * weight[i] for i in range(18))
```

### 5. 数据覆盖率惩罚

```python
final_score = raw_score * (indicator_count / 18)
```

指标缺失（如亏损股无 PE、无股息）会按比例降低总分。
