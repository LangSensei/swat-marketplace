# 环保板块价值评分公式 V2 (Environmental Protection Sector Value Score Formula)

## 概述

本公式为A股环保板块（东方财富板块代码BK0728）量化价值评分体系。
评分范围0-100分，分数越高表示该股票越具有投资价值。

**数据概况：**
- 覆盖125只有效标的（原始136只 - 4只B股 - 7只ST股）
- PE中位数35.6（仅正PE），PB中位数2.19
- 32只亏损股（25.6%），13只利润增速被±200%封顶
- 评分均值49.0，中位数50.1

**核心设计理念（PPP/环保行业特征适配）：**
1. ROE(12%)为主锚指标 — 伟明环保22.2%领先，区分运营型和工程型企业
2. 估值类(20%)标准配置 — PE 7%+PB 5%+股息率 8%；环保龙头高分红（洪城4.75%、永兴4.27%、军信4.26%）
3. 成长性(17%)略低于标准 — 项目收入确认周期长，PPP合同期内收入增长受限
4. 现金流(15%)从标准12%提升 — PPP运营商面临政府应收账款风险，现金流质量是核心风控指标
5. 风险类(8%)从标准10%下调 — 高杠杆是PPP/BOT结构性特征（D/A常>60%），不宜过度惩罚
6. 质量类(10%)中毛利率变化(5%)+速动比率(5%) — 毛利率趋势区分运营型和工程型
7. 18个非零权重指标参与回测，100%权重覆盖
8. V2规则全套应用：增长率±200%封顶、ROE/ROIC≤60%封顶、PE<0排除、覆盖率惩罚

**行业特点：**
- 125只股票（排除4只B股+7只ST），涵盖垃圾焚烧/水务/环境修复/监测/园林等子行业
- 32只亏损股（25.6%），工程/咨询类亏损较多
- PE中位数35.6（仅正PE股票），PB中位数2.19
- 13只利润增速被±200%封顶（10.4%），0只营收增速被封顶
- 0只ROE被60%封顶，0只ROIC被60%封顶
- 垃圾焚烧/水务运营商（高毛利40-55%、稳定分红）vs 环保工程/咨询（低毛利15-25%、收入波动大）分化显著

## 第三部分：公式

```
Value_Score = base_score * coverage_penalty
base_score = Sum(weight_i * score_i) / Sum(weight_i_participated)
score_i = industry_rank_percentile (0-100)
  越低越好: score = 100 - percentile
  越高越好: score = percentile
  percentile = (rank - 1) / (N_valid - 1) * 100

V2 rules:
  ROE cap: min(ROE, 60%)
  ROIC cap: min(ROIC, 60%)
  Growth cap: clip(growth, -200%, +200%)
  PE: loss-making (PE<0) excluded from PE ranking
  Coverage penalty: if indicators < 10, score *= (count / 18)
  Missing: excluded from that indicator, weight redistributed

Weight table:
  PE:         7% (low好)    PB:        5% (low好)    股息率:     8% (high好)
  ROE:       12% (high好)   ROA:       3% (high好)   ROIC:       5% (high好)
  毛利率:     7% (high好)   净利率:     3% (high好)
  营收增长:    7% (high好)   利润增长:  10% (high好)
  CF/净利润:   6% (high好)   FCF收益率:  5% (high好)   CF/营收:    4% (high好)
  资产负债率:  3% (low好)    流动比率:   2% (high好)   带息负债率:  3% (low好)
  毛利率变化:  5% (high好)   速动比率:   5% (high好)
```
