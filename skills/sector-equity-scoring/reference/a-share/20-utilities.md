# 公用事业板块价值评分公式 (Utilities Sector Value Score Formula)

## 概述

本公式是A股公用事业板块（申万一级：公用事业，东方财富板块代码BK0427）量化价值评分体系。
评分范围0-100分，分数越高表示该股票越具有投资价值。

**核心设计理念：**
1. ROE(12%)为主锚指标 — 公用事业ROE分化显著（新奥股份19.4%、华能国际19.0% vs 大连热电-26.6%），高ROE代表持续盈利能力
2. 股息率权重(9%)为行业最高 — 公用事业是A股典型高分红板块（蓝天燃气10.35%、长青集团6.61%、百川能源5.92%、华能国际5.74%），高股息率是价值投资核心安全边际
3. 成长性权重(13%)适度降低 — 电力/燃气/水务为成熟稳定行业，有机增长有限，降低权重避免过度惩罚稳定期公司
4. 现金流权重(13%)标准偏高 — 基础设施型资产，现金流稳定可预测，CF/营收(4%)权重提高反映行业特征
5. 风险权重(12%)标准 — 公用事业D/A普遍偏高（基础设施投资需要），D/A 4%+流动比率4%+带息负债率4%均衡分配
6. 质量权重(11%)偏高 — 速动比率(6%)权重提高，资本密集型行业即时偿债能力重要
7. 18个非零权重指标参与回测，100%权重覆盖
8. 评分规则全套应用：增长率±200%封顶、ROE/ROIC≤60%封顶、PE<0排除、覆盖率惩罚

**行业特点：**
- 136只股票（排除3只ST+4只B股后），涵盖电力(火电/水电/风电/光伏)/燃气/热力/环保能源等子行业
- 13只亏损股（9.6%），远低于煤炭(40.6%)和钢铁(23.3%)，反映公用事业盈利稳定性
- PE中位数25.9（仅正PE股票），PB中位数1.79
- 8只利润增速被+200%封顶，2只被-200%封顶（合计7.4%）
- 0只ROE/ROIC被60%封顶
- 高分红特征突出：蓝天燃气10.35%、长青集团6.61%、百川能源5.92%、华能国际5.74%、佛燃能源5.67%
- 29只股票股息率>3%（21.3%），体现行业分红文化

## 第三部分：公式

```
Value_Score = base_score * coverage_penalty
base_score = Sum(weight_i * score_i) / Sum(weight_i_participated)
score_i = industry_rank_percentile (0-100)
  越低越好: score = (1 - (rank - 1) / (N_valid - 1)) * 100  (rank ascending)
  越高越好: score = (1 - (rank - 1) / (N_valid - 1)) * 100  (rank descending)

Scoring rules:
  ROE cap: min(ROE, 60%)
  ROIC cap: min(ROIC, 60%)
  Growth cap: clip(growth, -200%, +200%)
  PE: loss-making (PE<0) excluded from PE ranking
  Coverage penalty: if indicators < 10, score *= (count / 18)
  Missing: excluded from that indicator, weight redistributed

Weight table:
  PE:         7% (low好)    PB:        5% (low好)    股息率:     9% (high好)
  ROE:       12% (high好)   ROA:       3% (high好)   ROIC:       5% (high好)
  毛利率:     5% (high好)   净利率:     5% (high好)
  营收增长:    6% (high好)   利润增长:   7% (high好)
  CF/净利润:   5% (high好)   FCF收益率:  4% (high好)   CF/营收:    4% (high好)
  资产负债率:  4% (low好)    流动比率:   4% (high好)   带息负债率:  4% (low好)
  毛利率变化:  5% (high好)   速动比率:   6% (high好)
```
