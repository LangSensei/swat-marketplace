# Textile Fiber Reference

## Use When

- You need to identify a woven synthetic fiber from lab clues (burn, DSC, density, FTIR, solubility)
- You need to match a fiber to its compatible dye class and recommended process window
- You need to compare cost and energy profiles across dyeing/finishing pathways

## Files

| File | Description |
|------|-------------|
| `fiber-identification-quick-reference.md` | Lookup table for identifying PA6, PA66, PET, PBT, CDP, PTT, and spandex from burn behavior, DSC peaks, density, FTIR bands, and solubility tests |
| `fiber-dye-matching-and-process-windows.md` | Maps each fiber family to compatible dye classes (disperse, acid, reactive) with indicative temperature, pH, time windows and auxiliary requirements |
| `cost-and-energy-structure-reference.md` | Comparative cost and energy data for common dyeing and finishing process pathways — use when evaluating commercial burden trade-offs |

## Reading Rules

- Treat all numeric values as indicative ranges, not guaranteed setpoints.
- DSC and density values are for oriented fiber; amorphous or drawn variants may shift by 5-15 units.
- Dye process windows assume standard atmospheric or HT jet equipment; beam or pad routes may differ.
- Cost data are relative indexes, not absolute prices — use for ranking, not quoting.
- When multiple fibers match a clue, use the elimination logic described in the identification reference rather than picking the first match.
